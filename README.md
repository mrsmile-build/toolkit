# ToolKit — Free Online Utility Tools

A collection of fast, clean, mobile-friendly utility tools built with plain HTML, CSS, and JavaScript.

## Tools Included

- **Word Counter** — Count words, characters, sentences, and reading time
- **Age Calculator** — Find exact age in years, months, days + birthday countdown
- **Loan Calculator** — Monthly payments, total interest, and cost breakdown
- **Percentage Calculator** — Solve any % problem: basic, increase, decrease, difference
- **QR Code Generator** — Generate QR codes for URL, text, email, phone, WiFi, SMS

## Tech Stack

- Pure HTML, CSS, JavaScript — no frameworks, no backend
- Hosted on GitHub Pages (free)
- Fonts: Google Fonts (Syne + DM Sans)
- QR library: qrcodejs (CDN)

## Deployment

1. Push this folder to a GitHub repository
2. Go to Settings → Pages → Source: `main` branch, `/ (root)`
3. Your site will be live at `https://yourusername.github.io/repository-name`

## Custom Domain

1. Buy a domain from Qservers or Namecheap
2. Add a `CNAME` file with your domain name
3. Point your domain's DNS to GitHub Pages

## Monetization

- AdSense slots are marked in the code with comments — replace with your `<ins>` tags after approval
- Contact form: connect to Formspree or Web3Forms for free
